Cloud Infrastructure Lab

Azure Virtual Network with Subnet Segmentation & NSG Firewall Rules

Designed and deployed a production-style Azure Virtual Network with distinct Frontend and Backend subnets. Hardened the environment using Network Security Groups to enforce least-privilege access — permitting HTTP traffic to the frontend while restricting SSH and RDP access to a single trusted home IP address.

Key Outcomes

  • Created a VNet with isolated Frontend and Backend subnets
  • Deployed two lightweight Linux VMs into separate subnets
  • Configured NSG inbound rules: HTTP allowed, SSH/RDP restricted to home IP only
  • Verified VM-to-VM connectivity across subnet boundaries
  • Documented network topology, IP schema, and NSG rule rationale

Enterprise Network Simulation

Multi-Site Enterprise Network: VLANs, OSPF Routing & ACL Security

Simulated a multi-department enterprise network in Cisco Packet Tracer connecting a Head Office and a Branch Office. Implemented VLAN segmentation for HR, IT, and Guest traffic, enabled inter-VLAN routing via Router-on-a-Stick, deployed OSPF/EIGRP for dynamic routing, and applied ACLs to restrict guest access to internal servers.

Key Outcomes

  • Configured HR, IT, and Guest VLANs on managed switches with 802.1Q trunking
  • Implemented Router-on-a-Stick for inter-VLAN routing on a single subinterface
  • Deployed OSPF and EIGRP to propagate routes between HQ and branch networks
  • Applied extended ACLs blocking Guest VLAN access to the internal server subnet
  • Tested and verified full connectivity matrix with Packet Tracer simulation
